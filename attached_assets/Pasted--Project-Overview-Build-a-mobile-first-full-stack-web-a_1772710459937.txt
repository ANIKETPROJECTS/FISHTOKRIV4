🎯 Project Overview
Build a mobile-first, full-stack web application for FishTokri (fishtokri.com) — an online fresh fish, seafood & meat retailer based in Mumbai, delivering across Thane, Mulund, Airoli, Bhandup, Vikhroli, Powai, Vashi, and Nerul.
The app has two distinct surfaces:

Customer-Facing Storefront — a beautiful, app-like experience for clients to browse daily availability and place order requests
Admin Panel — an internal tool for employees to manage inventory, availability, and product listings (full CRUD)


🛍️ PART 1: CUSTOMER STOREFRONT
Design Direction

Mobile-first (primary device is smartphone)
Aesthetic: Fresh, coastal, vibrant — think ocean blues, coral reds, clean whites with bold typography
Feel: Like a premium fish market app — energetic, fresh, trustworthy
Font pairing: Bold display font (e.g. Playfair Display or Fraunces) + clean body font (e.g. DM Sans)
Use animated availability badges (🟢 Available / 🔴 Unavailable / 🟡 Limited Stock)
Hero section with "Today's Fresh Catch" headline that updates daily

Home / Storefront Page

Product cards showing:

Product name
Category icon/emoji
Availability status badge (Available / Unavailable / Limited)
Price (optional, if admin sets it)
"Add to Order" button (disabled if unavailable)


Filter/search bar to find specific items

🔧 PART 2: ADMIN PANEL
Access

Separate route: /admin or subdomain admin.fishtokri.com
Protected by login (username + password, JWT auth)
Role: Employee / Manager

Dashboard Home

Summary cards: Total Products | Available Today | Out of Stock | Pending Orders
Quick-toggle section: flip availability of items in one tap
"Today's Update" status — shows last time inventory was updated + who updated it

Product Management (Full CRUD)
Product List View

Table/card list of all products
Columns: Name | Category | Status | Price | Actions
Bulk actions: Mark all in category as Available / Unavailable
Search + filter by category or status

Add New Product

Form fields: Product Name, Category (dropdown), Description (optional), Price (optional), Unit (e.g. per kg, per piece), Image upload (optional), Initial Status
Save to database

Edit Product

Edit any field
Change availability status: Available / Unavailable / Limited Stock
Set custom "limited stock" note (e.g. "Only 2kg left today")

Delete Product

Soft delete (archive, not permanent) with confirmation modal

Stock / Availability Management

Daily availability toggle panel — designed for quick morning updates
Category-level bulk toggle
Individual item toggle with status: 🟢 Available | 🔴 Unavailable | 🟡 Limited
Optional: set restock time/date

Order Management

View incoming order requests: customer name, phone, items, delivery area, time
Mark orders as: Pending → Confirmed → Out for Delivery → Delivered
Filter by status, date, area
Click to call / WhatsApp customer directly
